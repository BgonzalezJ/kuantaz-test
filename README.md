# Kuantaz test

Proyecto basado en laravel versión 12

## Requisitos

- PHP >= 8.2
- Composer
- Servidor web local o Herd

## Instalación de dependencias

1. Clona este repositorio:
   ```bash
   git clone https://github.com/BgonzalezJ/kuantaz-test
   cd kuantaz-test

2. Instalación de dependencias:
   ```bash
   composer install

3. Copiar variables .env:

Se debe copiar las variables BENEFITS_ENDPOINT, FILTERS_ENDPOINT y FILES_ENDPOINT del env.example al .env

   ```bash
   cp .env.example .env

## Documentación
